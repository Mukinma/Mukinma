## About me

I am a software engineering student from Mexico focused on building software that feels clear, polished and useful.

My strongest interest is not only writing code, but turning ideas into complete experiences: structured architecture, functional interfaces, visual identity, product logic and real user flows.

I like projects where design and engineering are not separated. A good system should work correctly, explain itself clearly and feel intentional from the first screen.
